Part 1: Responsive Web Designs

The first example of using responsive website will be max width property. This will ensure that whe images are being used, it will not stretch past the width limit. Two good reasons for this option. The first is that it will not outstretch the image to fill up the rest of the webpage. The second is that it will scale down if the image has to to fit onto the smaller devices. Folowing that step we are going to use the picture feature when if the browser is at a certain size, its either going to show the larger image or smaller one.
